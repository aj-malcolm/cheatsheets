## Other

### Dictionaries

Print all keys:
```py
for key in my_dict:
  print(key)
```

Print all values:
```py
for x in thisdict:
  print(thisdict[x])
```

Print all values:
```py
for x in thisdict.values():
  print(x)
```
  
print all keys and values:
```py
for x, y in thisdict.items():
  print(x, y)
```
